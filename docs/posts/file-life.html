File Life

Welcome to File Life–a touring project and travel blog about letting go of files. Follow along as a traveling fileman wanders.

As I travel, I collect files that people want to let go of. I ask each participant to delete the file they've given me, so that the only remaining copy exists on a USB I carry. In return, I take their files on a tour of the mountains and the oceans. When the USB fills up, I will delete the files, along with this website. You could think of this website as a memorial to files.
