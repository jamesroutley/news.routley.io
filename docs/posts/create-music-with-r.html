Show HN: Create Music with R

Create music with R.

Example

library(gm)

music <- 
  Music() +
  Meter(4, 4) +
  Line(c("C5", "D5", "E5", "F5"))
  
show(music)

Installation

Install gm:

install.packages("gm")

# Or install the development version from GitHub
pak::pak("flujoo/gm")

Install MuseScore. MuseScore is open source and free notation software.
