The Optics Library is designed to expedite the creation of publication-quality optical diagrams. Examples created with this library appear on Wikipedia:

  • anisotropic terahertz microspectroscopy (ATM)
  • terahertz time-domain spectroscopy (THz-TDS)

Notes:

  • Objects are transparent .png images. You may select and individually download the images below, or download the entire current archive using the Image archive (.zip) link above.
  • Different object orientations are provided for ease of use and to help maintain correct angular orientation of objects.
  • A PowerPoint file is provided as a link above that includes:
    • a fully editable library of individual objects and commonly used optics blocks (with a few additional objects like beam-lines and images in development),
    • tutorials to demonstrate how to create a diagram using this library in PowerPoint,
    • how to save the diagram as a .png image file for publication,
    • examples of full system diagrams.
  • Please direct any suggestions, issues, and requests to Tim (TJ) LaFave Jr.
