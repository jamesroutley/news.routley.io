post one - January 2022 - finartcialist

Happy new year 2022! We'll make it through.

In this post: a few words about the new edition of the zine mouvements sous-jacents and some news about the escarcelle project.

This new edition of the zine, in French, talks about the main works related to the finartcialist project in 2021; a short story on ultradeterminism; two texts explaining my creative process; a call for proposals for the next edition of the zine, the theme being économie: un jeu de société; and a bibliography summing up works discovered this last year that are about finance and art.

Here is a list of all the formats available for this new edition of the zine:

  • HTML page
  • .pdf for screen
  • .pdf for printing

If you have any question or comment, more specifically about the call for proposals for the next edition, please let me know by email: info@finartcialist.com

escarcelle, le marché aux sentiments - beta version

The beta version of escarcelle, the feelings market has launched.

The goal of this first public step of the project is to create a marketplace for bartering postcards. No other object will be accepted for now.

For more information, please read the blog of the project!
