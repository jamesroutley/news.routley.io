Installation

Prerequisites

  • Docker
  • Docker Compose

Steps

# Clone the repository
git clone git@github.com:goniszewski/grimoire.git

# [RECOMMENDED] Update the `.env.docker` to set the initial admin user credentials

# Build and run the containers
docker-compose up
