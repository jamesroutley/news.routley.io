PSA IV

Every once in a while I get messages about my blog posts. As I don't have comments enabled on my blog, most of these messages come by e-mail. While this is perfectly acceptable and prudent, I'd like to share and alternative way to communicate with me today: Github issues!

This blog is entirely open source, and I've talked about the tech before. I keep some of my longer-term ideas in my head by filing Github issues tagged as reminder.

Fig. 1: A typical reminder.

If you want me to write about something or check if it's on my radar already, I'd suggest you look at the list of issues or file one yourself.
