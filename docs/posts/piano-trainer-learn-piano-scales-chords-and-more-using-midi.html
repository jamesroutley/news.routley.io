Run Locally

You'll need to setup Rust and Tauri CLI by following the Getting Started guide here.

$ git clone https://github.com/ZaneH/scale-trainer.git
$ cd scale-trainer
$ yarn && yarn tauri dev

Build target binary

Outputs to /src-tauri/target/release/bundle
