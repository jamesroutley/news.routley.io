Minimum bipartite matching

$$ P^{\star} = \underset{P \in \mathbb{P}}{\mathrm{argmin}} \left( P C \right) $$

The assignment problem is a special case of the larger family of optimal transport problems, which mathematicians have looked at since the 18th century.

We can rewrite the assignment problem such that the optimization variable ranges over the Birkhoff polytope; this rewritten form is equivalent to the original one since the cost function is linear in the argument P, so we expect the optimum to lie at a vertex of the admissible region ğ”¹ (i.e.Â to be a permutation matrix).

$$ P^{\star} = \underset{P \in \mathbb{B}}{\mathrm{argmin}} \left( P C \right) $$
