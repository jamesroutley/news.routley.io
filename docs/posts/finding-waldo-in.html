Finding Waldo in π

The SMBC comic Convert included as the “Votey button” a challenge to extract a Waldo visual from the digits of π.

I present waldo.gif (here blown up to 25x size):

The pixel data in this gif are the 23,074,248th through 23,075,235th hexadecimal digits of π! (Equivalently, the 184,593,977th through 184,601,880th bits).

Here’s a little Python script that will verify that for you! First head over to pi2e.ch to download the first billion hexadecimal digits of π in the file pi_hex_1b.txt. (Or get the hexadecimal digits from a source of your choice!)

You also need the image manipulation package Pillow.

from PIL import Image

waldo = Image.open('waldo.gif')
pixdata = waldo.tobytes().hex()

with open('pi_hex_1b.txt') as fid:
    fid.seek(23074249) # 1 more to skip initial "3."
    pihexits = fid.read(988)

print(pixdata)
'1c688fc9a5ba54a16f1f…'
print(pihexits)
'1c688fc9a5ba54a16f1f…'
print(pixdata == pihexits)
# True!
