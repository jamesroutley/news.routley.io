Computability and Compressibility

Home Blog

02 Oct, 2023

Is the statement "most numbers are incompressible" equivalent to "most numbers are incomputable"?

A compressible number is one that can be reduced to a smaller number by some algorithm. The algorithm as a whole must be smaller than the original number.

A computable number is one that can be computed by some algorithm. It would not do to label a number "computable" if the only algorithm to compute it were longer than itself, because the algorithm could be replaced by that number. So computable numbers must be compressible.

Conversely, compressible numbers, since they are expressed by an algorithm, are computable.

So these two seemingly disparate concepts, computable and compressible, describe the same set of numbers.

Thinking about it further, I think I have this wrong. 2 is incompressible because there is no shorter representation, but it is not incomputable, it can be computed in all sorts of ways. All compressible numbers are computable, but not all computable numbers are compressible. In addition there are some numbers, like Chaitin's constant, that can be described simply but cannot be computed. I am not sure if these count as compressible, they are not algorithmically compressible but in some sense logically compressible.
