About SQLook

SQLook is a modern web-based SQLite database manager with a nostalgic Windows 2000 interface. It combines the power of contemporary web technologies with the classic aesthetics of one of computing's most iconic eras.

Features

Database Management

  • Open and create SQLite databases
  • Export databases and query results
  • Visual database structure viewer
  • Real-time database statistics

Query Tools

  • Interactive SQL query editor
  • Query history tracking
  • Query formatting tools
  • Results export (CSV, JSON, SQL)

Table Generator

  • Visual table creation interface
  • Pre-built table templates
  • Sample data generation
  • Foreign key support

Query Helper

  • Common SQL templates
  • Interactive examples
  • Syntax highlighting
  • Context-aware tooltips

Technical Details

Built with: HTML5, CSS3, JavaScript, SQL.js

Database Engine: SQLite 3
