Resetting the timer in my toothbrush

December 19, 2023

There is an NFC chip inside of my toothbrush head, and it counts the time. After 360 minutes of work it will signal that it needs to be replaced, even if it doesn’t really. It’s possible to reset the counter, by sending a message from a phone to the toothbrush head. It’s all a little insane.

  1. Scan the toothbrush head with the NFC Tools app. Copy the serial number.
  2. Find the part number written on the bottom of the brush head.
  3. Enter the data into the password generator and copy the generated NFC command.
  4. In NFC Tools, go to Other -> Advanced NFC Commands. Paste the command into the input and send it to the toothbrush head.
