Review: CS50P Introduction to Programming with Python

Started: January 2025

Overview: CS50P is Harvard University’s introduction to programming with Python, available free as OpenCourseWare. From the course welcome:

Learn how to read and write code as well as how to test and “debug” it. Designed for students with or without prior programming experience who’d like to learn Python specifically. Learn about functions, arguments, and return values (oh my!); variables and types; conditionals and Boolean expressions; and loops. Learn how to handle exceptions, find and fix bugs, and write unit tests; use third-party libraries; validate and extract data with regular expressions; model real-world entities with classes, objects, methods, and properties; and read and write files. Hands-on opportunities for lots of practice. Exercises inspired by real-world programming problems. 

Each week of the course includes a video lecture taught by David J. Malan, section, and shorts, as well as a problem set. If you register for the course through edX, you can submit the problem sets for feedback and receive a certificate after completing the course.

My Story: During my first week at the Recurse Center, I decided to prioritize improving my Python skills. Since I loved the CS50x course, I decided to work through CS50P.

The CS50P lectures don’t have the quite the same energy that the CS50x lectures have. Instead of a live audience of students attending in person, we see a few puppets watching the lecture instead. It seems like the lectures were given live to a remote audience, since there are audio questions from students from time to time.

That said, David Malan’s teaching is consistently engaging and clear and the problem sets were both fun and challenging. The online notes are a great resource, and frequently include links to the official Python documentation, which I tried to use as much as possible. The early weeks of the course were a good review and opportunity to practice for me, since I had some experience with Python already from CS50x, but later on in the course I learned things like how to write unit tests and work with regular expressions that were new to me. I found the regular expressions problem set to be the most challenging for me in the course.

For my final project I created a program called Bookmark Analyzer, a command-line program to sort links and remove duplicates from a bookmarks export TXT or HTML file.

I can definitely recommend CS50P to anyone who is new to Python. I still have more to learn but I think this course gave me an excellent foundation for continuing to program in Python.
