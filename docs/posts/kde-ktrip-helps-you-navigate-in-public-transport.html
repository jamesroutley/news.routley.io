KDE KTrip – helps you navigate in public transport

KTrip helps you navigate in public transport. It allows you to find journeys between specified locations, departures for a specific station and shows real-time delay and disruption information.

This button only works with Discover and other AppStream application stores. You can also use your distribution’s package manager.

KTrip Windows installers are also available to download from the binary-factory. These versions don’t contain the translations and are intended for testing purpose. Get involved and help us make them better!

  • Stable version. Generated from the latest version of the stable branch. Contains the latest bugfixes.
  • Unstable version. Generated from the latest version of the development branch.
