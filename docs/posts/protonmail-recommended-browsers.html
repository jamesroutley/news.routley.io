Protonmail Recommended Browsers

Proton Mail has been fully tested and works well with most modern browsers. However, we only recommend browsers that also respect your privacy. These include:

  • Mozilla Firefox(new window)
  • Brave(new window)
  • Tor Browser(new window)

Learn more about the most secure browsers for your privacy(new window)

Proton Mail has also been fully tested and works well with:

  • Google Chrome(new window)
  • Safari(new window)
  • Microsoft Edge(new window)
  • Opera(new window)

Proton Mail is compatible with other browsers not listed here (for example Pale Moon(new window), IceCat(new window), and SeaMonkey(new window)) but has not been tested extensively using them. It is therefore possible you may encounter more bugs when using browsers that are not officially supported.

The browser you are using must support the Web Cryptography API(new window) in order to work with the encryption we use.

With all browsers, we always recommend using the latest stable version available.
