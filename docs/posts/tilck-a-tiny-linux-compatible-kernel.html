I discovered a very cool idea called the Old Computer Challenge just last week! It started in 2021 and runs for a week each year with some theme. It is a chance for folx to try new ways of approaching computing and to work with old/limited hardware and technology. There’s an IRC channel people hang out in and share ideas as well as a list of websites that people are logging their experiments and experience on too.

Suffice to say, it really resonated with me! Turns out that it started this week (July 13 - 19), so I’m just in time. :) For this week, I have some ideas that I’ll aim to do to participate and will blog about it under the occ2025 tag as I go:

  • Experiment with non-graphical enviroments for linux ( TTY only maybe??? :O )
  • Try using qutebrowser instead of firefox (mouseless!)
  • Look into TUI/terminal replacements for some of the last GUI programs I use still
  • Look into trying out Chimera Linux which has a unique, cool approach that uses Musl C + FreeBSD tools for userland! <3

⁂

This should be fun! :) I’m looking forward to the excuse to play around with my computer and setup a bit this week and to see what others get up to as well. Cheers!
