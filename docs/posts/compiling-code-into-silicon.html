Compiling code into silicon

Compile code into silicon

Process scaling is coming to an end and it is a social imperative that we find a new path to extend the Moore's Law exponential. The most viable option is extreme silicon specialization, which will require fast automated translation from program to silicon. Compiling simple programs into silicon should be like using llvm or gcc: fast, automated, and accessible.

For designs that are too complex to be compiled at the command line,  development should be like programming in Python: creative, productive, and fun.

SiliconCompiler is an open source compiler framework that automates translation from source code to silicon using a standardized compiler data Schema, a Python object oriented API, and a distributed systems execution model. Read the Documentation for the full story.
