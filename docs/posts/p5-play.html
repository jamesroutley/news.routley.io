P5.play

p5.play is a p5.js library for the creation of games and playthings. Our goal is to make coding accessible for artists, designers, educators, and beginners. If you are not familiar with p5.js, you should start here.

It uses p5.js for graphics rendering. We share the Processing Foundation's goal of make coding more accessible for artists, designers, educators, and beginners. Behind the scenes p5.play v3 uses planck (Box2D), a physics simulator that's been used in popular 2D games like Angry Birds and Tiny Wings.

Ready to learn how to use p5.play to prototype, develop, and publish games? Version 3 will feature major improvements over v2 , it is in late beta testing and will be released in August 2022. Look at some of the editable examples on this website. When you're ready to start your own game you can use this starter template sketch.
