This is a richer way to display your hacker news bio. To use it:

  1. To opt-in, paste "{your username}.at.hn" anywhere in your profile's 'about' section.
  2. Go to https://{username}.at.hn?refresh.
  3. Between HN/our caching, you might need to wait a couple minutes.
  4. Use markdown to make things prettier. Submit PRs on github if you like.
  5. Note: Outgoing links are rel=nofollow unless you're >200 karma (anti spam).

See an example: padolsey.at.hn.
