Show HN: Editable Games

What is this?

You can edit/change the games in the list below. You can upload your own images, and even re-program the game play using SCL.

Once you've made your new game, you can have it activated to run on a domain of your choice by visiting the Publishing page. For example, if you have an account a itch.io, you can activate your game for your itch page.

This is a demo of how editable and customizing games work using Canvas Language. If you choose a game below, you'll be taken to a dev studio where you can edit your game and download a project file, for re-upload later.
