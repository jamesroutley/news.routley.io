Cheat Engine contains GPL-licensed code but disallows commercial use

@ThisGitHubUsernameWasAvailable - misapplying a license does not change the license of the whole project, it just means the project is violating the license. Licenses are not "viral" in that sense.

If someone accidentally includes GPL in their code they have two options-

  1. Update their project to a compatible license. This doesn't have to be GPL (it could be MIT for instance), but it has to be a compatible license.
  2. Remove the GPL code.

Most companies opt for the second option.

In this case it looks like the issue is in TCCLib, not Cheat Engine specifically. It also looks like it may just be a mislabeled file. Either way this issue should be mentioned to the upstream library, and once they fixed the issue the downstream programs should update. If TCCLib doesn't fix the problem then exploring replacements is also a good idea.
