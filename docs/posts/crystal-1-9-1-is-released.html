Crystal 1.9.1 Is Released

We are announcing a new patch release of the Crystal 1.9 series with two bugfixes.

Pre-built packages are available on GitHub Releases and our official distribution channels. See crystal-lang.org/install for installation instructions.

Stats

This release includes 2 changes since 1.9.0 by 2 contributors. We thank all the contributors for all the effort put into improving the language! ❤️

Changes

The changes are fixes for minor regression bugs in the standard library and library.

For more details, visit the changelog.

If you find any further issues, please report it.
