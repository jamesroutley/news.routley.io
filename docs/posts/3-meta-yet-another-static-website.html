3. Meta: yet another static website

Jacob Vosmaer's blog

2019-12-07

I have an aversion to using proper static website generators so this site yet another one-off hand-rolled project based on ERB and Markdown.pl.

The hardest part about building this is getting the S3 / Cloudfront hosting right. Because making Cloudfront changes is so slow (10+ minutes) it gets annoying to do things by trial and error.

But then it works and you can forget about it.
