Subject: Eglot has landed on master
Date: Thu, 20 Oct 2022 14:14:29 +0100 
Hi all,

As of commit 83fbda715973f57dc49fe002d255ecaff8273154, Eglot has landed
on Emacs' mainline.

Eglot's manual in Texinfo format is also in included in the merge and
lives in doc/misc/eglot.texi.  A published version of this manual in
HTML format can be consulted at https://joaotavora.github.io/eglot/

Over at elpa.git, the GNU ELPA package "Eglot" will now be produced from
the file lisp/progmodes/eglot.el.  The upcoming version of the Eglot
ELPA package will be 2.0.

Over at the GitHub site, I've updated the README.md with the notice of
the change of upstream.

Let me know if I missed any steps or if the information provided in any
of these sources is inconsistent or misleading.

Thanks to everybody that helped build Eglot to this point.

João
