SkillzTruth - Proof of concept demonstrating how to cheat on Skillz apps

The App

The tool runs on a Windows machine that is connected to the same network as your iPhone. Unfortunately, there is a bit of setup required to get things working.

This app allows you to do the following in any Skillz-based game:

  • Modify your game score to whatever value you want.
  • Makes you a chat moderator.
  • Allows you to ban any user from the app.
