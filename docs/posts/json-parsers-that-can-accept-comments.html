JSON parsers that can accept comments

I know what you've been told. Douglas Crockford said that JSON can't have comments. And you accepted that. Understandable mistake; we've all been there.

But I'm here to tell you there's another way, because Douglas Crockford is not your dad. You can put comments in JSON files if you know they're going to be read only by a reasonable parser. If you maintain a parser, you can make it accept comments and nothing bad will happen!
