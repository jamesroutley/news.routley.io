Discover a new chapter in UNO’s story

The first UNO started a new world for DIY and making. In fact, it started a new dimension.

Now, this world is expanding with UNO R4.

Set a new standard

UNO R4 preserves the standard form factor, shield compatibility, 5 V power supply and robustness of the UNO range while offering no less than a 32-bit Arm® Cortex®-M4 and a 16-fold increase in memory. Thanks to enhanced thermal design, the UNO R4 boards can be powered up to 24 V. 

Wi-Fi and Bluetooth capabilities

Featuring an ESP32-S3-MINI coprocessor that enhances the capabilities of its RA4M1 microcontroller, with Wi-Fi and Bluetooth® Low Energy connectivity, this board makes it easier to connect to the internet and create your IoT project

12x8 on-board LED matrix

If you are working on a creative project using animations or plotting sensor data, this is ideal for you because it doesn’t require any additional hardware.

Qwiic I2C connector

The board includes a Qwiic I2C connector to facilitate quick prototyping. Thanks to a wide variety of compatible modules that can be connected over I2C.

Diagnostics for runtime errors

The board features a built-in mechanism to detect operations that might cause the board to crash, such as division by zero, before they cause any trouble. When it catches an error, the board provides you with a detailed explanation.

Support for battery-powered RTC

The UNO R4 features additional pins so you can turn off the main microcontroller while keeping the internal RTC powered with an external battery.

Arduino Cloud Compatibility

The Arduino UNO R4 WiFi is fully compatible with the Arduino Cloud, so for example you can store data remotely, build interactive dashboards and upload sketches Over-The-Air.

More memory and faster clock

With a 16x increase in memory and 3x the clock speed of previous versions, you can leverage the UNO R4 Minima to perform more precise calculations and handle more complex and sophisticated projects.

USB-C Connector

The Arduino UNO R4 boards get an USB-C® connector, that is mechanically robust an comes in small size.

New on-board peripherals

The Arduino UNO R4 features several exciting new peripherals, including a 12-bit DAC, CAN BUS, OP AMP, and SWD port to expand your maker potential, allowing you to easily tackle increasingly advanced projects.

HID over USB-C

You can simulate a mouse or keyboard when connected to its USB-C port. You’ll be able to create a cool interface with minimal effort and in no time!

Larger voltage range and electrical robustness

Also thanks to enhanced thermal design, the UNO R4 boards can be powered up to 24V. Paired with their robustness, this makes them a great choice for your next project. The RA4M1 microcontroller has overcurrent protection on the pins.
