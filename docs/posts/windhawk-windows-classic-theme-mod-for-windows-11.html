Windhawk Windows classic theme mod for Windows 11

If you are using the Windows 10 taskbar, install the OpenShell utility. After installing the OpenShell, put the patched file StartMenuDLL for OpenShell 4.4.190 into the Open Shell folder, replacing the existing one. Alternatively, install the patched version of OpenShell4.4.191

Alternatively to patching OpenShell you can install the mod Classic Taskbar background fix, but read carefully the instructions there.
