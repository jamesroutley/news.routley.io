Immortel VAR

The Immortel family of serif typefaces comprises four variants (Infra, Colera, Vena, and Acedia), all designed with multiplexed set widths to be interchangeable without causing text to reflow. The Infra and Vena variants – inspired by Robert Granjon and Jacques-François Rosart, respectively – are intended for use in running body text. They include “Median” italics for longer blocks of text, as well as more exaggerated italics for stronger emphasis. They both also offer a variable grade axis for slight weight adjustments, to accommodate different typographic environments and user preferences.

Tags: body text, serif, ball terminals, multiplexed, multiplexed weight, oldstyle
Design:Clément Le Tulle-Neyret
Publisher:205TF
Characters:Latin
Released:2021-04-14
Licensing:Trial, Paid/commercial
Info/fonts:205.tf

Golos Text VF & Golos UI VF

A pair of related sans typefaces commissioned by Smena and AIC Media for official state and social service websites in Russia. Golos Text is designed for continuous reading, whereas Golos UI – with slightly narrower proportions and multiplexed weights – is intended more for user interface elements.
