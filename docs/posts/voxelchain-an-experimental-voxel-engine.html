FAQ

- Which hardware does VoxelChain run on?

The minimum required hardware we're targeting is the NVIDIA GTX 1060 and the Apple M1

- What does VoxelChain refer to?

A core feature of our engine is that you can infinitely "chain" worlds together and digitally communicate logic between them: Worlds inside worlds inside worlds..

- What kind of simulation is VoxelChain?

Our simulation engine Weltenjaeger allows to visually design complex voxel behaviour with basic electric components, which then gets simulated on a massive scale

- Real-time ray tracing in the browser, how?

We're using WebGL and have spent a lot of time optimizing our rendering engine. We use a variety of custom and modern rendering algorithms
