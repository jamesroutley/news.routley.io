Show HN: SingleFile is finally available on Safari (macOS/iOS)

SingleFile is a Safari extension that helps you to save an entire webpage including images, styling, frames, fonts etc. as a single HTML file.

Nov 12, 2022

Great for saving web pages before they disappear

I have been using SingleFile for many years in my desktop browsers and I am very pleased to have it in iOS Safari now too. iOS 16 also seems capable of displaying the locally saved HTML files, which is convenient.
