README.md

sqlite-gui is a lightweight SQLite editor for Windows powered by C++/mingw64/WinAPI.

Download the latest version

Features

  • Syntax highlighting and code completion
  • Edit multiple tables at the same time
  • Colorized data grid
  • Store an execution history and user scripts
  • Export/Import data
  • Database diagram
  • Database comparison
  • Search text in the whole database
  • Quick data references
  • Terminal mode
  • Charts
  • Query parameters
  • Query shortcuts
  • Custom functions
  • Jinja scripting (experimental)
  • Data generator
  • Extension pack
  • Built-in REST API web server
  • Demo database "Bookstore" for beginners
  • Does not require installation

Need a much simpler tool? Try sqlite-x!
