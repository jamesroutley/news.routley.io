Simple blocks magically transform into polished designs.

Noya is the culmination of everything we learned from building design tools for companies like Airbnb, now available to everyone. It combines the freedom to iterate in wireframes with the power of design systems.

Devin AbbottNoya co-founder/CEO, previously worked on Airbnb Design Tools

Noya makes design as easy as wireframing, freeing you to focus on what matters most.

Wireframing enables exploring different directions without overly investing in design details for a single direction before validating it’s the right one.

Noya breathes new life into wireframing by combining wireframes with a design system to generate high-fidelity designs in real time. This means you can stay in wireframing mode longer, be more creative, and automatically produce design outputs in parallel that excite everyone around you.

Select a design system

The design system determines the block types and styles. Chakra UI is currently available, and more design systems and custom themes are planned.

Wireframe with blocks

Click and drag to draw a block. Then pick a type and define the content with the simple text-based interface. Review the generated design and adjust.

Export design files and code

Generate what your team needs, from a rasterized mockup to vector layers importable to Figma, or even React code as a starting point for your frontend.

Frequently asked questions

Can designers/engineers/product managers/entrepreneurs use Noya?

Yes. If you have an idea for a website, web app, or mobile app, you can use Noya to draw wireframes of your idea. Noya will automatically generate designs and code for your wireframes based on a design system.

Noya is designed to work across functions and workflows. Product managers and entrepreneurs can use Noya to quickly create impressive mockups to pitch a product idea. Designers can use Noya to keep the flexibility and agility of wireframing while producing high-fidelity artifacts. Engineers can use Noya to rapidly design a first interface for their feature and generate React code as reference.

We'd love to hear about how you're using Noya. Join our community on Discord and chat with us!

Does Noya work with my existing design tools?

Noya outputs designs in multiple formats: a shareable web link to preview the design online, PNG, SVG, PDF, and React code. We provide specific exports for Figma and Sketch.

Our export feature is currently in beta and will require some adjustments to make it pixel perfect. We recommend testing it early in your design process so you can know to expect.

What design systems are supported?

Noya currently supports Chakra UI. This design system is based on the open-source Chakra UI component library, combined with our own layouts and rules to create blocks in Noya.

We plan to add support for additional design systems as well as theming and customization.
