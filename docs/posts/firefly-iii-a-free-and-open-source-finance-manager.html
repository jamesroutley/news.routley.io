More

"Firefly III" is a self-hosted financial manager. It can help you keep track of expenses, income, budgets and everything in between. It supports credit cards, shared household accounts and savings accounts. Itâ€™s pretty fancy. You should use it to save and organise money.

You can read more about the way of working, the features and the current state in the documentation ("read more").

Full Transaction Management

Firefly III features a double-entry bookkeeping system. You can quickly enter and organize your transactions in multiple currencies.

The tool allows you to add unlimited accounts, transactions and meta-data. It works surprisingly well with many years of financial records in the database. Firefly III can also manage crypto currencies.

You can read more about this in the documentation of Firefly III and you can try Firefly III for yourself on the demo website .

Budgets, categories and tags

Name your poison. Do you like to work with tags? Need to budget your expenses? Want to categorize all of your hobby expenses? Look no further. Firefly III supports all kinds. Budgets can be expanded with limits in multiple currencies, so you can budget both your daily household expenses and what you spend in Imperial Credits when visiting Tatooine.

You can read more about this in the documentation of Firefly III and you can try Firefly III for yourself on the demo website .

Rule Engine

No more tedious transaction entering. Use rules to quickly convert shorthands to detailed transactions. Clean up your banks abysmal CSV files. Use the rule engine to automatically set budgets, remove categories, append descriptions, change notes or something else entirely.

You can read more about this in the documentation of Firefly III and you can try Firefly III for yourself on the demo website .

Import transactions

A special Data Importer helps you import data into your Firefly III administration.

It supports many (thousands!) of banks and financial institutions, but it can also import from CSV files.

Reports

Firefly III has advanced reporting capabilities, showing your expenses per week, month or year. But it can also help you audit your accounts with detailed list views. Or perhaps compare budgets or tags? It's all possible.

You can read more about this in the documentation of Firefly III and you can try Firefly III for yourself on the demo website .

JSON REST API

For the advanced and mobile users out there, Firefly III features an expansive JSON REST API that allows you to tap into Firefly III's most important features. Create transactions, manage categories and get chart data from any app or system.

You can read more about this in the API documentation of Firefly III , where you can try the API against the the demo website .
