Portable Web Documents – An Alternative to PDF Based on HTML5 (2019)

Portable Web Documents are a technology similar to PDFs (Portable Document Format) implemented in Polar which support offline caching of full HTML documents, and with (in the future) improved support for video, charts, and other compelling features.

Polar uses PWDs and PDFs to manage the users reading and allows the user to keep all documents in a central repository and allows for suspend/resume of reading, tagging, and annotation.
