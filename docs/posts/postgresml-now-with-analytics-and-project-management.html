Train models

Train multiple models from a Postgres table or view, with the best model automatically selected through quick and painless iteration.

					
CREATE VIEW user_orders AS
SELECT
	users.user_id,
	products.price,
	users.total_purchases,
	products.id AS product_id,
	orders.status
FROM products
JOIN orders
ON products.id = orders.product_id
JOIN users
ON users.user_id = orders.user_id;

SELECT * FROM pgml.train(
	'Buy it again',
	'regression',
	'user_orders',
	'status',
);

Deploy to production

Make predictions in SQL and use them for filtering and sorting inside queries, simplifying ML integration in your stack.

					
SELECT * FROM products
JOIN user_orders ON
products.id = user_orders.product_id
ORDER BY pgml.predict(
	'Buy it again',
	ARRAY[
	  user_orders.user_id,
	  user_orders.price,
	  user_orders.total_purchases,
	  user_orders.product_id
	]
) DESC
LIMIT 25
