ARJ Software

Please explore our world of PC file data compression by browsing the site of ARJ Software.

Our archivers offer more than just data compression. They also offer powerful file management functions including version management as well as reliability and data damage protection for advanced PC users.

Our ARJ32 archiver supports Windows 7, Windows Vista, Windows XP, Windows 2000, Windows NT, Windows 95/98, and Windows Millenium.

ARJ is a file archiver that compresses files into a disk space saving archive. It is fast, reliable, and has a rich set of features which include Windows 9x, Millenium, 2000, XP, Vista, and 7 long filename support, file version management, data damage protection, configurable command options, multiple volume archives (archives that span removable media) and much more.
