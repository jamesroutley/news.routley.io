Added SingleFile and files.gallery to bookmarks

Published on Oct 29th, 2024.

I’ve discovered a couple of small projects: SingleFile and files.gallery that merited inclusion in my bookmarks section: Projects & companies.

SingleFile:

A browser extension (and CLI utility) that converts complex webpages into self-contained html files for archival purposes. The 2024 archive.org takedown made me realize that, for pages that I might want to reference far into the future, it’s safer to archive things on my own rather than rely on a third party. SingleFile makes this easy and (hopefully) outputs to a durable file format that I can rely on in the long term.

files.gallery:

A single-file PHP application that you can drop into a directory on a web server which creates a rich listing of the files within. I’ve ended up purchasing a license because this is by far the best option I’ve found for self-hosting galleries of photos to share with friends/family.
