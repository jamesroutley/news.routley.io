Show HN: Donut Browser, a Browser Orchestrator

Create as many local browser profiles as you need. Each profile is completely isolated with its own settings, extensions, and data.

Built-in proxy support for all browsers except TOR Browser.

Download and manage multiple browsers including Chromium, Firefox, and more. Switch between browsers seamlessly.

Optimized for performance with minimal resource usage. Quick startup times and smooth operation.

Forget about opening links in the wrong browser. Donut Browser will allow you to choose what browser you want to use for each link.
