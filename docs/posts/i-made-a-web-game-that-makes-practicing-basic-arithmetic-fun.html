Show HN: I made a web game that makes practicing basic arithmetic fun

Mathic is a magical math search game that'll test your speed and accuracy!

Find an equation that = the Magic Number (M.N.)

You can see the current Magic Number above the grid:

HIGHLIGHT equations by tapping

When you find the correct equation, tap the start of it and then the end of it:

Equations can be highlighted horizontally and vertically, either backwards or forwards.

Solve as many Magic Numbers as you can before the game ends!

Build streaks for higher scores

The more Magic Numbers you solve in a row, the higher your streak will grow! A higher streak will lead to higher scores, so watch that accuracy!

Stuck? Use a skip!

To use a skip, just tap the current Magic Number:

You only get 3 skips per round, so use them wisely!

Good luck!

This is a fast math game, so you only get 60 seconds each round. Try to score as high as possible! The clock starts as soon as you solve the first Magic Number.

Customize your game! You can enable dark mode, change the game operators, modify the grid dimensions, and more in the game's settings.
