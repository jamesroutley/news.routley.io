Every possible Wordle solution visualized

We all have daily rituals to keep us focused, create structure, and provide some enjoyment. For a lot of us, Wordle has worked itself into that rotation. As I've been playing over the last year, I kept finding myself asking questions like, "how many possible words are there?", "how many words start and end with the same letter?", and "how many words contain X?". To satisfy my own curiosity, I built out this Wordle visualization tool.

You can click around on different letters to see the words that use that letter in that particular position. I hope you enjoy playing with it as much as I did building it!
