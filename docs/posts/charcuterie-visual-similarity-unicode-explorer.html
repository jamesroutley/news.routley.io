Charcuterie – Visual similarity Unicode explorer

✎

A visual explorer for Unicode. Browse the character set, discover related glyphs, and learn more about the scripts, symbols, and shapes that make up the standard.

To power visual similarity, rendered glyphs are embedded with and compared in vector space.

Charcuterie is still under active development, and feedback is always welcome. If you would like to support the project and help make more time for it, consider a donation.
