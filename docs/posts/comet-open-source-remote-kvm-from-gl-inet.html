Comet open source remote KVM from GL-iNet

Portable Hardware Design

Comet is a compact and powerful Remote KVM product offering 2K resolution at 60FPS for smooth and clear video output. Powered by a 1.5GHz quad-core processor, it features a Gigabit Ethernet connection for fast network speeds, along with USB 2.0 for external extensions, such as connecting a finger switch. It also supports a Type-C power supply, HDMI-IN for video input, and a USB-Device port for connecting a mouse, making it a versatile and efficient solution for remote device management.

Comet is compatible with the following devices: Mini PCs (e.g., iMac), desktop computers, laptops, servers, industrial PCs, and TV boxes (video decoding devices).
