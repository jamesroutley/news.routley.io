Show HN: My demo for vector embeddings for the Earth's surface

This site uses a deep embedding model to build a vector DB representation of the world hosted which can be queried via pgvector. This interface shows how to perform a spatial similiarity search to look up locations nearby with similar patterns of land use / land cover.

How to Use

  • Move the circle around to query different areas. The circle's center is the query point.
  • Shrink or expand the circle for a smaller or bigger search area. Only sites within the circle will be included in the search
  • Adjust the coverage fraction slider to tweak the sensitivity.
