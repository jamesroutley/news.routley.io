Meet Methodable is a series of articles about Methodable, the human-programming tool. Each article focuses on one section of the Meet Methodable interactive guide and provides additional discussion.

The ideal human-programming tool will make following guides so straightforward that there is zero onboarding time to the guide itself: any person, with no context, can click a link and immediately start making progress in a workflow that’s completely new to them.

Methodable is on its way to that ideal state, but in the meantime there’s a tiny bit of onboarding (about 2 minutes).
