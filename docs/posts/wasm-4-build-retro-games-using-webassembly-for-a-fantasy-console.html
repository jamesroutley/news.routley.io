WASM-4: Build retro games using WebAssembly for a fantasy console

A quick video introduction to WASM-4.

Minimalist and Easy

Everything you need to build complete games without being overwhelming. Get creative within 160x160 pixels and 4 colors.

Language Agnostic

Use any programming language, as long as it can compile to WebAssembly. AssemblyScript, C/C++, Rust, Go, and more.

Run Anywhere

Besides great desktop/mobile web support, run your game natively (no webviews!) even on low-powered microcontrollers and obsolete hardware.

Multiplayer

Fast online multiplayer support with rollback netcode based on GGPO.

Community

WASM-4 is supported by a diverse and friendly community of creators who love building small.
