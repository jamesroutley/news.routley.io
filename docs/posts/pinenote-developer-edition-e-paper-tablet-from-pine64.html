PineNote Developer Edition – E-Paper Tablet from Pine64

Description

The PineNote is one of the world’s most advanced e-paper tablets. The device features a 1404 x 1872 resolution panel with a fast refresh rate and 16 grayscale levels. The Wacom EMR and capacitive layers are protected by hardened scratch-resistant glass with an anti-glare coating.  The device features a frontlight with 36 levels of white or amber illumination settings.

Powered by the fast and efficient RK3566 SoC with four A55 cores, backed by 4GB of RAM and 128GB of eMMC storage, this is a capable yet portable device capable of taking notes, writing and reading documents, reading books, browsing the internet, and terminal usage — all at the same time.

This first production run of the PineNote will be great to write software for, but not great to write notes on. The device ships without an operating system with flashing mode enabled. As a result, this batch is strictly for developers.

Developers will need to use a coupon code to complete the purchase.
