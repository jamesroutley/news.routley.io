ScreenPlay Is an Open Source Live-Wallpaper App for Windows and OS X

ScreenPlay is an Open Source cross platform app for displaying Wallpaper, Widgets and AppDrawer. It is written in modern C++20/Qt6/QML. Binaries with workshop support are available for Windows & OSX (soon Linux/KDE) via Steam.

Wallpapers

ScreenPlay supports a vast variety of Wallpaper. Most video formats as well as HTML5, Qml and Gif Wallpaper are supported.

Widgets

Tired of opening multiple apps for important information? ScreenPlay desktop Widgets are the solution, that allow you to have little helper apps like CPU usage, time, weather or the latest reddit threads on your desktop.

Apple Silicon

The latest software like Qt6 allows us to have Apple Silicon support! This includes working Universal Binary support. ScreenPlay.app will work with both Intel and Apple Silicon at full speed.

Easy Qml/Js Scriping

QML is a easy to learn declarative language, to create custom Wallpapers and Widgets. QML is a powerful language with native ECMA 2017 support, that allows for the creation of interactive and animated content.

SysInfo API

Display of system information, such as CPU usage, memory usage and IP-Address. This can be especially useful for power users, developers and sysadmins who need to monitor their system stats.

Live Reloading

ScreenPlay code hot reloading feature enables fast iteration, allowing you to see the changes you make to your projects in real-time. This feature eliminates the need to constantly restart your content, saving you time and helping you stay focused on your creation.

Weather Forecast API

Another great addition to ScreenPlay expanding feature set is the new Weather API. It is build around open-meteo.com, that will provide a detailed 7 day forecast. We now additionally include Weather Icons for nice set of matching icons.

One Click Import And Export

ScreenPlay makes it super easy to export and import your creations with just one click. By simply dragging a .screenplay file into the app, you can quickly and easily import your favorite wallpapers and widgets, saving you time and hassle.
