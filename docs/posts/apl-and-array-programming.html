APL and Array Programming

Update: We’ve already done some study group sessions, and you can watch the recordings. You’re welcome to join any time, even if you haven’t been in previous sessions.

This is for discussion of programming in array languages such as APL, J, BQN, and K, as well as array-oriented features of other languages and notation such as einstein notation, numpy broadcasting, mathematica’s threaded. We may even get into discussions of combinatory logic…

We’re doing informal study sessions, initially focussing on APL, at 10am Tues-Fri Brisbane time (here’s the current time in Brisbane – note that it’s one day later than US, so it’ll be Mon-Thurs US time). You can join here:

Minor tips:

  • Once you finish a Dyalog APL session in Windows, you may need to switch back to the normal keyboard setting to make the Ctrl key work again
  • To start all arrays from 1 (the default) :
  • To start all arrays from 0:
  • In Dyalog, click to turn on/off boxes around each element
  • Adding a ⊢ before a variable assignment allows you to display the variable without an additional line (A ← 3 assigns 3 to A vs ⊢A ← 3 assigns 3 to A and prints out the value in A)
  • to have a high minus ¯3, use backtick 2 (or ctrl-2) and enter 3
