Internet Independent Wireless Mesh Communication App

qaul is an internet independent wireless mesh communication app. Communicate peer-to-peer with the people around you, without any Internet or communication infrastructure.

Automatically detect other qaul users nearby, broadcast public messages to everyone, create chat groups and send end to end encrypted chat messages, images & files.

Communicate directly from device to device via your local wifi network, or via the shared wifi network of your phone. Mesh local clouds together via manually added static nodes. Use this peer to peer communication method to communicate internet independently and completely off-the-grid.

Public Broadcast Messages

Send a message to everyone connected.

Encrypted Chat

End-to-end encrypted chat messages & group-chat.

Send Files & Photos

Send files and photos via the chat messenger.

User Discovery

Users in the same Wifi are automatically discovered. All interconnected users are displayed and reachable.

Network Overview

See who is connected and reachable

Delay Tolerant Network

Send messages to users who are not currently connected.

The app has been translated to many languages and can be easily translated to new ones.
