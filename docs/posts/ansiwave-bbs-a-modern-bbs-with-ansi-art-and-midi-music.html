Ansiwave BBS, a modern BBS with ANSI art and MIDI music

Run the terminal client without any arguments to begin browsing the BBS. See the README on Github and the screencast for other things you can do.

If you want to try browsing the BBS offline, just clone the board with git:

git clone http://bbs.ansiwave.net/boards/kEKgeSd3-74Uy0bfOOJ9mj0qW3KpMpXBGrrQdUv190E/.git

...then point the terminal client to the directory:

ansiwave kEKgeSd3-74Uy0bfOOJ9mj0qW3KpMpXBGrrQdUv190E

That's it. You are now browsing the board without any network connections, and everything (including search) still works. The next time you `git pull` you'll have the latest posts.
