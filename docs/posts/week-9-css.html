Week 9: CSS

Over the last week, I’ve really been learning a lot about CSS. Josh Comeau’s course continues to be very useful — I especially appreciate the little games he put in to help cement tricky concepts like margin collapse and how absolute and fixed positioning interact with containing blocks. I’m realizing that there are some remarkably tricky and subtle concepts in CSS!

So far, I’ve made it all the way through the flow layout module and most of the way through the positioned layout module. I’ve learned so much already, even before making it to flexbox or CSS Grid, which I know are the more modern tools that people use more today — I’m excited to get to them! The sandboxes where you recreate a website mockup have helped me cement that learning, too. You can see the first workshop I did here (don’t forget to click the “Project View” icon in the upper-right to switch the sandbox to “Current Module View”, which actually shows the styling).

I’m not quite at the point where I want to start skinning my app yet, but I will be next week for sure. As I said, there’s just so much to CSS I never realized was there, so it’s taking a bit longer to learn than I thought it would!

Overall, my strategy for Recurse from last week is still working as well. In category theory, I’m still learning a lot and it’s still interesting: natural transformations are gradually getting firmer in my mind, I’m starting to understand adjunctions, and it was pretty cool to see the recursive descent parsing I learned about in Crafting Interpreters being used for category theory too! And for socializing, I had to cut down coffee chats this week from 5 to 3, but after doing that, I feel like I’m finally hitting a sustainable pace here at Recurse.

One more thing: I found out about !!Con through the Recurse Center (apparently many of the organizers and speakers have gone through RC), so I decided to go (online) last weekend. It was cool! Especially getting to make Game Boy music with the amazingly talented Chipzel using LSDJ! I’m continuing to learn about new things and have unexpected awesome experiences through Recurse, and as Thanksgiving approaches, I just want to say that I’m very thankful for that. ❤️

Next time: More CSS, and hopefully working on my Elixir / Phoenix app again!
