ECL Runs Maxima in a Browser

Raymond Toy

27 Jan 2025 27 Jan '25

4:06 p.m.

Thought you would be interested to know that Marius Gerbershagen has used ecl to compile Maxima to wasm to run in a browser. I believe it's a pretty complete implementation with fancy TeX display of formulas and nice graphics via gnuplot compiled to wasm. See it at http://maxima-on-wasm.pages.dev/ -- Ray
