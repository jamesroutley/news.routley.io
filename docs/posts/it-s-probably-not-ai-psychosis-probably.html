It’s probably not AI psychosis probably

This will be a confused post, because I am confused.

Everyone is confused. We’re all lost.

It’s good! It’s good to feel lost! The only way is through!

But also - dude! What the hell is going on.

I am truly only micro-posting this for two reasons: one very good friend said something so funny that I’m stealing it (with permission?) and posting it here:

“How can anyone be an AI expert if all the tools are 2 weeks old”

That’s funny! But also true. I was telling him - whoah, man, you’re doing extraordinary work with AI! Are you also kinda going crazy?! You’re writing this distributed database client - you’re doing amazing stuff! Do you want to talk about it? Let’s record a podcast!? Bro?

And his answer was: how can I even talk about this. What does anyone truly know about it. Every tool is so new - and so ever changing - how can anyone state they’re an expert.

Well, ok, he’s humble and smart, and he is not the type to sell an online AI class on how to boost your productivity, etc. Ok, fine. I accept, begrudgingly.

But I’m also going a little bit insane. And creating things I’ve never made before (at 1000x the speed)? And sometimes feeling like I’m going absolutely mad - sorry, I am, honestly, for sure - but then, at other times, I’m succeeding and it’s 100% true and real.

It feels like this, sorta, sometimes:

Stopping yourself at any point with these LLM tools feels like you were almost there but you ALSO know that you believed that kind of stuff in the past and kept digging and there were no diamonds at all.

But sometimes there are! A lot of diamonds! And you did go crazy in the middle (and believed it was all false) but then it worked! So you were wrong! You were not going insane you were just… ?????

Anyway. The second reason I’m posting this is because I’m excited about the next project I’ll be soon launching but I want to polish it for like a few more days. So - hey - excitement. Something new. Yay. A new greg technology project. Greg hasn’t gruesomely expired (I haven’t!), he’s still making stuff (I am!). Hopefully, that’s a good thing. I mean, of course it is. yay! life.

Miss you all. Be well, readers. Don’t go too too crazy. Touch grass, or at the very least, make an MCP server do so on your behalf…… xx
