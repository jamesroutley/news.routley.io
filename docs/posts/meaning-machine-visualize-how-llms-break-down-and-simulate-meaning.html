Tokenization (BERT-style)

Why Tokenization Matters

Language models do not process text as humans do. Instead they first break your input into tokens—the smallest fragments (words, subwords, or symbols) they know how to handle. Each token maps to a numeric ID in the model's vocabulary.

Subject–Verb–Object (SVO) Triples

About SVO Extraction

We use spaCy's dependency parse to extract simple (subject, verb, object) triples—i.e. who did what to whom—by looking for the main verb (ROOT/ccomp/xcomp) and its nominal subject and direct object.

POS Tagging (spaCy)

Why POS & Dependencies?

spaCy assigns each token a Part-Of-Speech (POS) tag, a dependency label describing its role (e.g., nsubj, dobj), and a head (the token it depends on). This reveals the grammatical skeleton of your sentence.

Embedding Visualization (DistilBERT)

How Embeddings Work

BERT maps each token into a 768-dimensional vector capturing its contextual meaning. We reduce those vectors to 2D using PCA so you can see clusters of similar tokens.

Dependency Role Glossary

Tap to expand
Label Name Description
nsubj Nominal Subject The noun performing the main action — e.g. “student” is doing “submitting.”
aux Auxiliary Verb A helper verb — e.g. “did” helps “submit.”
neg Negation Modifier A word that negates — e.g. “n't” negates “submit.”
ROOT Root The main verb or predicate — e.g. “submit.”
advmod Adverbial Modifier An adverb modifying a verb or clause — e.g. “on time.”
prep Prepositional Modifier Introduces a prepositional phrase — e.g. “on.”
pobj Object of Preposition The noun in a prepositional phrase — e.g. “time.”
det Determiner Introduces or limits a noun — e.g. “The.”
attr Attribute Describes subject after linking verb — (not used here).
amod Adjectival Modifier An adjective modifying a noun — e.g. “young.”
compound Compound Modifier A noun modifying another noun — e.g. “final.”
cc Coordinating Conjunction Connects equals — e.g. “and,” “but.”
conj Conjunct The second element joined by a cc.
ccomp Clausal Complement A clause as object — e.g. “I think [he left].”
xcomp Open Clausal Complement A clause without its own subject — e.g. “to leave.”
mark Marker Introduces a subordinate clause — e.g. “that.”
