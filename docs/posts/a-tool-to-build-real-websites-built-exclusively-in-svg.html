Show HN: A tool to build real websites built exclusively in SVG

A new kind of site

What you create in Adobe Illustrator is what you see online — instantly

  • a single page
  • a full website
  • existing content— posters— brochures— reports

Svija is a website builder based on Adobe Illustrator.

Svija is lightweight and fast. Using Illustrator, you’ll be able to create rich, detailed web pages that are simply impossible with other website builders.

With Svija, you spend all your time doing the fun part, designing, and there’s no conversion or coding! And, Svija is free.

Step 1: Start Synching

Svija Sync is a Mac app that synchronizes your Illustrator files with your website. Each time you start a design session, you’ll just launch Svija Sync and press Upload. The synchronization process will continue until you press Pause or quit the program.

Step 2: Design

We’re guessing that you already understand this step.

Step 3: Changes are uploaded automatically

When you’re ready to save your document, just press Save — Svija Tools saves the Illustrator document and also an SVG file in a special folder.

The SVG file is immediately synchronized with your website by Svija Sync.

Your page is live, instantly.

Step 4: Manage

Svija Admin is a web interface to: - choose the address of your web pages - optimize for search engines - mix and combine Illustrator files - create custom headers, footers, and other reusable modules - all placed to the pixel with incredible precision

Included with Svija Admin is free hosting for your content.

Since the Illustrator files from your computer are also kept on the server, your content is continuously backed up!
