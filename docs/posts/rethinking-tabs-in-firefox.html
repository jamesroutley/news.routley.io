Show HN: Rethinking Tabs in Firefox

Grasshopper is a Firefox extension for tab management. It has 4 item modes: Tabs, Stars, Closed, and History. Tabs display all your open tabs, and you can interact with them. Tabs are orded by last visit date. Your freshest tabs are always at the top. And if there are too many, you can type to find it. Stars are like bookmarks that you save manually. Stars can be used to open websites easily. Closed are recently closed tabs. History searches and filters browser history.

There's a Settings window with various options.

The background and text colors can be customized. Random dark and light themes can be generated.

There's useful buttons like "Playing" and "Clean". "Playing" focuses the tab that is emitting sound. So you can return easily to what you were watching or listening to. "Clean" closes all the tabs which are not pins or playing. So if you have many tabs open, you can hit "Clean" and keep only the important ones. One way to find items quickly is typing something and if doesn't appear hit [Tab] to search in Stars. The [Tab] order can be customized in Settings.

Items can be filtered by type. Like if they're pinned, playing audio, muted, etc. Secure = https websites Insecure = http websites
