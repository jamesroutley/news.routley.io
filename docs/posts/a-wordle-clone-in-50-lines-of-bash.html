It's still not quite right. As of the 6th revision, if the secret word is LEMON and the guess is WEAVE, the second E is yellow but it should be grey.

Here is one way to fix it:

remaining="" output=""
for ((i = 0; i < ${#actual}; i++)); do
    if [[ "${actual:$i:1}" != "${guess:$i:1}" ]]; then
        remaining+=${actual:$i:1}
    fi
done
for ((i = 0; i < ${#actual}; i++)); do
    if [[ "${actual:$i:1}" != "${guess:$i:1}" ]]; then
        if [[ "$remaining" == *"${guess:$i:1}"* ]]; then
            output+="\033[30;103m ${guess:$i:1} \033[0m"
            remaining=${remaining/"${guess:$i:1}"/}
        else
            output+="\033[30;107m ${guess:$i:1} \033[0m"
        fi
    else
        output+="\033[30;102m ${guess:$i:1} \033[0m"
    fi
done
printf "$output\n"

Explanation:

The variable remaining tracks the letters of actual that can contribute toward yellows.

P.S. There is an easy way to shave a couple of lines: instead of having a separate piece of code for displaying all-green when the guess is correct, you could just let the main case handle that output.
