Build Your Own Flight Sim in C++ (1996)

When Retro City Rampage was ported to DOS, the developer lamented how he had not only thrown out all of his DOS programming/gamedev books, but how none were online. Say no more.

This is a book on DOS game programming with the goal being to write your own flight simulator with SB16 sound. Included is a CD-ROM image with example code.

ISBN: 1-57169-022-0
