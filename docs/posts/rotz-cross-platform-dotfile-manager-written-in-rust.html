Rotz: Cross platform dotfile manager written in Rust

Rust DotfilemanagerRust Dotfile managerRust Dotfile sRust Dot sR ust DotsR otsRot sRotz 👃

Fully cross platform dotfile manager and dev environment bootstrapper written in Rust.

⬆️⬇️

Keep dotfiles up to date

With Rotz you can easily sync your local dotfiles with a git repository using the rotz sync command.

🚀

Deploy dotfiles

You can automatically symlink your dotfiles to the correct places using the rotz link command.

💿

Install software

You can bootstrap your new machine using the rotz install command.

⚙️

Versatile configuration

You can specify where to link your dotfiles to and what software to install in yaml, toml or json config files.

🪟🐧🍎

Cross platform

Rotz works on Windows, Linux and MacOs and has full support for different configurations on each platform.

🦀

Open source and written in rust

If you find a bug or have a feature request feel free to open a github issue or even a pull request.
