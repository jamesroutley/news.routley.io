Charts.css: CSS data visualization framework

Quick Start GitHub Repo

Charts.css is a modern CSS framework. It uses CSS utility classes to style HTML elements as charts.

The source code is available on GitHub (opens new window). If you like the project, please consider to star the repo on GitHub.

Semantic Structure

Uses HTML tags to display data

Customizable

Custom styling using CSS

Responsive

Scaled using media queries

Accessible

Raw data inside the markup

Large Variety

Many supported chart types

No dependencies

Zero external overhead

Small Library

72kb file size, 6kb gzipped

Better Performance

No JS = no rendering
