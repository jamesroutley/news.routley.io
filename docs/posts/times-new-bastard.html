Times New Bastard

Download Times New Bastard

It's Times New Roman but every seventh letter is jarringly sans serif. Original idea by ommanyte on Tumblr:

Times New Bastard is a modified version of Nimbus Roman No. 9 L and Nimbus Sans, original source here.

Using it on the Web

Turn on ligatures and optimiseLegibility:

font-feature-settings: "kern" on, "liga" on;
text-rendering: optimizeLegibility;

Using it in Microsoft Office

Turn on Ligatures.
