X210Ai is a new motherboard to upgrade ThinkPad X201/200

X210Ai is a new motherboard to upgrade Thinkpad X201/200, The details as below:

1, CPU: Ultra 7 165H, Ultra 9 185H;

2, Storage support: Two of M.2 SSD PCIE 4.0 (one 2280, one 2242), the original 2.5inch SATA;

3, Memory: DDR5 5600MHz max to 128G (64G +64G);

4, Two type-c: one is support thunderbolt 4.0, other one is full function type-c;

5, Support Output HDMI 2.1;

6, Display: support screens same as X2100, such as the original X201/200’s display, 13inch 3000×2000, 13.3inch 1920×1200 and 13.3 inch 2560×1600;

7, WWAN 4/5G;

8, The postion of SATA can install the 2rd fan;

9, A new daughterboard;

10, The X210Ai project currently cannot guarantee coreboot support, but we remain committed to exploring possible solutions.
