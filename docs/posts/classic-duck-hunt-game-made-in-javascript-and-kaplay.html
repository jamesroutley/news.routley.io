Classic Duck Hunt game made in JavaScript and Kaplay

I wanted to remake the classic Duck Hunt game in JavaScript using the KAPLAY library. So here is Duck Hunter, a Duck Hunt-like.

~ CONTROLS ~

- MOUSE/THUMB to aim.

- CLICK/TAP to shoot

Only Available on PC : You can press the "P" key on the keyboard to pause the game.
