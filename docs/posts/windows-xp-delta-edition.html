Windows XP Delta Edition

Welcome to Windows XP Delta Edition!

Windows XP Delta Edition is a modification of Windows XP which aims to recreate the Windows XP Beta 2 aesthetic and bring back lost features, functions, and programs from previous versions of Windows, along with prerelease versions of Windows XP.

Customization and Extras

Windows XP Delta Edition's main look and feel is Whistler Luna, but there are many options for you. Out of the box, you get Whistler Luna, Watercolor with 5 color schemes, RTM Luna with its original color schemes, and Windows Classic; all of which contain little fixes and touch-ups.

Old Programs, Modernized

Windows XP Delta Edition also brings back programs and functions which either were removed from or were left unused, complete with a facelift to fit Windows XP's user interface. These include the Program Manager, the Welcome applet, Windows Media Player 6.4*, Windows Media Player 95*, and more.

*The Post-install Script must be run after setting up Windows Media Player 9 Series to get these to show on the Start Menu.

More Consistency

Windows XP Delta Edition is much more clean and tidy under the hood, replacing bitmaps, icons, and other resources which do not fit Windows XP, giving Windows XP Delta Edition a familiar, yet consistent look and feel.
