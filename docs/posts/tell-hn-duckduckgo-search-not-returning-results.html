Tell HN: DuckDuckGo Search Not Returning Results

For the last several hours, DuckDuckGo has not been returning search results. Rather, after a brief delay, it returns only the text: >There was an error displaying the search results. Please try again.

Unsurprisingly, this appears to be affecting both mobile and desktop search.

AFAICT, there hasn't been an official announcement from DDG although there is a reddit thread[0] just noting the above information.

Hopefully DDG will be back soon, as I'm loathe to go back to Google for search.
