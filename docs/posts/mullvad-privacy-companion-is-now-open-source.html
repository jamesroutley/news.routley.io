Six months ago, we released Mullvad Privacy Companion, a free Firefox extension helping to improve your online privacy.

Today, we're happy to announce a new version, now open source!

The Mullvad Privacy Companion interface has been reworked around recommendations, making it easier to see what you can do to improve your browser privacy.

Why is it still in beta?

There are still some bugs to be fixed and some features we want to implement. Before releasing it as stable, we want to get the code audited.

We welcome your feedback! Email us at support@mullvad.net or open an issue on our issue tracker*.
