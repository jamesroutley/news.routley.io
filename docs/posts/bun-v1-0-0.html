Bun v1.0.0

Today is an exciting day.

To install Bun v1.0:

curl https://bun.sh/install | bash
# or you can use npm
# npm install -g bun

To upgrade to Bun v1.0:

Read Bun v1.0.0's release notes on Bun's blog
