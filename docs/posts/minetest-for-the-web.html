Minetest for the Web

This is an open source, experimental port of Minetest to WebAssembly using Emscripten.

Development is still in the early stages (project started 3/15/2022). Major unresolved issues:

  • No saved files. Worlds and mods are lost when page is closed or reloaded.
  • Network play is through a websocket proxy. This adds some lag.
  • 'Browse Online Content' is broken on Firefox/Safari (but 'Join Game' should work)
