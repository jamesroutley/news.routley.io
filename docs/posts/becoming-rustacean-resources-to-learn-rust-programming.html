Becoming Rustacean: Resources to Learn Rust Programming

The Documentation Book

This book teachs how to make awesome documentation for your crate. 

The standard Rust distribution ships with a tool called rustdoc. Its job is to generate documentation for Rust projects. On a fundamental level, Rustdoc takes as an argument either a crate root or a Markdown file, and produces HTML, CSS, and JavaScript.

A Dedicated Book on the Rust compiler

When you want to familiarise yourself with the knobs available in the Rust compiler, this book is the one.

Rustc is the compiler for the Rust programming language the project provides. Compilers take your source code and produce binary code, either as a library or executable. 

Rust Error Codes Index

This one has an in-depth explanations of the errors you may see from the Rust compiler.

The Unstable Book 

The Unstable Book consists of a number of chapters, each one organised by a "feature flag." 

Awesome Rust 

A curated list of Rust code and resources from applications written in Rust to databases, cryptography and distributed system implementation of Rust.The list also includes established and startup companies currently adopting and using Rust as one of their programming arsenals.

This same repo has a list of highly useful resources to start with programming in Rust.I will be sharing some of the resources below. But I suggest you go about them one by one on the actual Github repo.

Awesome Rust Streaming

This is a community-curated list of livestreams about the programming language Rust.

Exercism: Want to learn and master Rust?

Join Exercism’s Rust Track for access to 103 exercises grouped into 13 Rust Concepts, with automatic analysis of your code and personal mentoring, all 100% free.

Idiomatic Rust

This repository collects resources for writing clean, idiomatic Rust code.

Rust for professionals

This is for you if you have very sound programming experience in systems programming.

Rust Gym

This one has Rust solutions of different types that are useful in solving different programming problems.

Rust. How do I start?

This one is collaborative advice for this casual question that gets asked many times, so here it is as a Github repo anyone can contribute to and improve.

Rust-learning

This repo has links to blog posts, articles, videos and other materials for learning Rust.

Rustlings

This project contains small exercises to get you used to reading and writing Rust code. This includes reading and responding to compiler messages.

Rust CS

This is unique because it is a CS curriculum with a Rust flavour.

First steps with Rust

This one is a a Rust learning material from Microsoft.

CIS 198: Rust Programming

This class was taught at the University of Pennsylvania in 2016.

The Rustacean Station Podcast

A community project for creating podcast content for the Rust programming language.

Awesome Rust Books

This one has a list of books on Rust programming in the following categories: Starter, Advanced and Resources.

Learning Rust

This one is another sorted curated list of Rust learning resources hosted on GitHub.

The list is quite comprehensive in curating Rust learning materials that are perfectly suitable for anyone at a different level of competency in Rust programming.

Are we learning yet?

A work-in-progress to catalogue the state of machine learning in Rust.

Are we web yet?

Another work-in-progress to catalogue the state of web development in Rust.

Rust has mature and production-ready frameworks in Rocket, Warp and Tide. These provide everything you’d expect from a web framework, from routing and middleware to templating and JSON/form handling. Like they say, "these days there are crates for everything web-based".

For a more comprehensive list of features that are whether here on Rust release yet or not, check out the Areweyet portal.

Rust Application Books

This one lists books on several Rust applications.The list includes books on:

  • Zebra, the Zcash Foundation's independent, consensus-compatible implementation of a Zcash node

  • using the Rust programming language with Espressif SoCs modules, a highly integrated ultra-low-power development board which includes Flash and PSRAM

  • Anachro, a Network Protocol, and a PC architecture for a microcontroller-based system.

  • Crosvm is a virtual machine monitor that is based on Linux's KVM

  • Various Rust architectures for reactive UI

  • Relm4, an idiomatic GUI library inspired by Elm and based on gtk4-rs. 

  • GUI development with Rust and GTK 4

  • nalgebra, a linear algebra library written using the Rust programming language.

  • Plotly.rs, a plotting library powered by Plotly.js. 

Rust Cookbook

This one is a collection of simple examples that demonstrate good practices to accomplish common programming tasks using the crates of the Rust ecosystem.

Unofficial Rust Books

Under the “The Little Book of Rust Books,” there is a second sub-category labelled “Unofficial” because they are maintained outside the Rust language community. In addition to books on the language, resources of materials are available for Applications and Macros.

Comprehensive Rust 

This is a four-day Rust course developed by the Android team. The course covers the full spectrum of Rust, from basic syntax to advanced topics like generics and error handling. It also includes Android-specific content on the last day.

Learn Rust With Entirely Too Many Linked Lists

In this series, the author will teach you basic and advanced Rust programming entirely by having you implement six linked lists.This series will teach you the following in the process:

  • The different pointer types in Rust

  • Ownership, borrowing, inherited mutability, interior mutability, Copy

  • All The Keywords: struct, enum, fn, pub, impl, use, ...

  • Pattern matching, generics, destructors

  • Testing, installing new toolchains, 

  • Unsafe Rust: raw pointers, aliasing, stacked borrows, UnsafeCell, variance

Criterion.rs

Criterion.rs is a statistics-driven micro-benchmarking tool. It is a Rust port of Haskell's Criterion library. Criterion.rs benchmarks collect and store statistical information from run to run and can automatically detect performance regressions and measure optimisations.

Learn by Running Algorithms

TheAlgorithms/Rust repository on Github has most of the basic algorithms written in Rust under the following categories: Backtracking, Big Integer, Ciphers, Compression, Data Structures, Dynamic Programming, General, Geometry, Graph, Math, Searching, Sorting, and String.

Using Unsafe for Fun and Profit

Given Rust's popularity and position as a systems programming language, you'll probably reach a point where you want to integrate a Rust module into some existing application. This guide was created to fill the current gap in knowledge when it comes to doing more in-depth Foreign Function Interface (FFI) tasks than simply calling one or two functions from a C library.

Futures Explained in 200 Lines of Rust

This book aims to explain Futures in Rust using an example-driven approach, exploring why they're designed the way they are and how they work. The book also takes a look at some of the alternatives we have when dealing with concurrency in programming.

Rust and WebAssembly

This small book describes how to use Rust and WebAssembly together.

Rust Fuzz Book

Fuzz testing is a software testing technique used to find security and stability issues by providing pseudo-random data as input to the software.

The Rust Performance Book

This book contains many techniques to improve Rust programs' performance—speed and memory usage. The Compile Times section also contains some techniques that will improve the compile times of Rust programs. Some techniques only require changing build configurations, but many require changing code.

Secure Rust Guidelines

Rust is a multi-paradigm language with a focus on memory safety. This guideline will help you see how to follow those guidelines to make a program worthy of Rust’s benefits in memory-safe software development. 

The Rustonomicon- Digging into the Dark Side of Unsafe Rust 

The Rustonomicon digs into all the awful details that you need to understand when writing Unsafe Rust programs.

Learn Rust With JetBrains IDEs

Another interesting tutorial I found, specifically anchored around Rust development environment configuration and tooling, is this one by a team at JetBrains.

Rust For Starters by Ashish Kapoor

This one is available on Youtube with a playlist comprising videos on the basic aspects of Rust programming. 

Tour of Rust

This is meant to be a step-by-step guide through the features of the Rust programming language. Rust is often considered a language with a steep learning curve. Still, the developer of this interactive online learning platform hopes to convince you there's much to explore before we even get to the complex parts.
