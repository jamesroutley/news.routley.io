What is Assimilate!?

Assimilate is a Flood-It/Drench-style game where you work on filling the board with one solid color. You only get a certain number of moves to begin with, and each level will give you one less move. Every level is procedurally created at run-time to provide you with the most unique experience every time!

The game features 4 custom color palettes (and the ability to edit them) so that color-deficient individuals feel included. In the October 2021 Edition (v1.1.0), users can now create and save their own palettes!

Most recently (v1.2.0), the game was updated to include a splash screen as well as many quality of life improvements. You can now change the view mode to a square, which removes the color buttons. In any view mode, you can make a move by clicking on any of the squares on the board!

New gameplay is on the way! The next version will introduce new actions which can rotate the board or explode a little chunk when activated.
