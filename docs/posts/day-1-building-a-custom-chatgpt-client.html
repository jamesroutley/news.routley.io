Day 1 - Building a custom ChatGPT client

2023-10-01

I use ChatGPT a lot. I would like a local record of all my interactions, and later on run a script to create daily summaries.

This seems like a good opportunity try out all the cool things I've read about: litestream, duckdb, pydantic, fastapi, modal, mojo.

I also like the idea of bootstrapping an OpenAI client by asking ChatGPT, a bit like writing a Go compiler in Go.

The very first step involves checking OpenAI API calls return what I expect.

It does not. Time to lazyweb.
