ANGULARIS

The slightly different stacker game.

Angularis is a Tetris-like stacker game, but with shapes based on triangles instead of squares. You can rotate shapes left and right and move them horizontally and diagonally This powers some crazy new moves.

Touch Controls

  • swipe left = move left
  • swipe right = move right
  • swipe down = move down
  • tap left = rotate left
  • tap righ = rotate right
  • swipe diagonally left / right = move diagonal left / right
  • swipe up = hold shape and replace with next shape

Keyboard Controls

  • arrow left = move left
  • arrow right = move right
  • arrow down = move down
  • a = rotate left
  • s = rotate right
  • d + arrow left / right = move diagonal left / right
  • arrow up = hold shape and replace with next shape
  • space = super hard drop
