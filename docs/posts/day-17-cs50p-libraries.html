Notes

  • Some concepts that are more clear to me now after today’s study/practice:
    • if __name__ == "__main__": is used so you can import a function from a program without running the main code from that program.
    • Some libraries (like sys, random, json) are built in to Python and just need to be imported with import. Others (like cowsay, requests) are third-party packages that need to be installed first, then imported.
    • Python requests.get() method can be used to send a GET request to a URL.
    • It’s important to try various inputs when testing code to make sure the problem is not the input.
