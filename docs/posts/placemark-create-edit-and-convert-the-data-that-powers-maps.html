Placemark: Create, edit, and convert the data that powers maps

Placemark is the one place to create, edit, and convert the data that powers maps, in real-time with your team.

Spatial is only part of geospatial data. Placemark handles the rest – attributes – with a particular pizazz. Table editing, editing JSON values, long text - we've got it.

Have a CSV file with coordinates? How about a GeoJSON file? Or some KML from Google Earth? Maybe you're drawing data from scratch. It'll all work great with Placemark.
