I built a game where you spend Elon Musk's money

DISCLOSURE: If you buy through affiliate links, I may earn a small commission. (disclosures)

TL;DR - A game where you get $1 trillion of Elon Musk's money and spend it.

Try it now: Spend Elon Musk's Money

What is it

  • You get $1 trillion of Elon Musk's money.
  • Spend it on a catalog of things from a $6 coffee to a $500B Mars colony - a Tesla, a mansion, buying Twitter, a Falcon 9 launch.
  • Your remaining balance updates as you go and a receipt tracks everything you bought.
  • When you're done, you can see how much his money grew in interest while you were shopping.

Why I built it

  • A billion dollars is hard to imagine, a trillion is almost impossible. Musk just became the first ever trillionaire and it was hard for me to visualize that so wanted to make it easier to conceptualize.
  • I've been wanting to build more tiny games and experiments since Recurse Center and this one felt timely and fits my atomic artifact.
  • It's the first TypeScript lab in hamy-games, so it sets a foundation for more (I typically use Rust).

How it's built

  • Frontend: TypeScript and Vite. Everything runs in browser w ~10kb.
  • Backend: Runs on my Rust and Axum labs server, the same one behind the GoodReads dashboard, using CloudSeed Rust.
  • Process: built agentically through my hamy changes pipeline. Images all AI generated with an adhoc generation pipeline to FAL AI.
