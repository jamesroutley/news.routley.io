Binary data analysis

Veles combines advanced hex explorer and data visualization features with an extensible framework for binary data analysis. Reverse engineering binaries? Exploring file system images? Steganography? Veles supports your work in all these fields.

Easily extendable

With the number of different architectures, file formats and encodings you can't always expect to find the tool you need online. Veles uses client-server architecture, where each analyser can run in a separate process. All you need to do is get the data from the server, implement a Python function to parse it and return the results.

Open source

We released an open source alpha version of Veles in late 2016 and will be releasing an updated and improved versions each month. We are willing to share our expertise with the community to drive the change in how we view working with binary data.

Continuously tested

We believe that Capture The Flag (CTF) competitions are an excellent testing ground for binary data analysis tools like Veles. The challenges met during the competitions are an endless source of unusual inputs and surprising edge cases. Our team uses Veles in all CTFs and we'd be glad to see other teams benefitting from this valuable tool.
