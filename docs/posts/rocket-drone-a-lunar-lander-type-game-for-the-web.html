Show HN: Rocket Drone, a lunar lander type game for the web

Rocket Drone is a simple yet challenging game where you must pilot a rocket-powered drone to the exit of each level, using only left and right thruster controls, while managing fuel consumption and avoiding collisions with walls and obstacles.

Demonstration

It is highly recommended that you go through the tutorial before starting free play or daily challenges.

The URL automatically updates on each level. Simply copy and paste to share particular levels. When in replay mode, the URL will also update to include the encoded replay data, so it too can be shared.

Rocket Drone is mobile friendly. To go fullscreen on iOS, use the 'Add to Home Screen' option and launch from the home screen.
