Calmira Reborn is a Windows 9x-inspired shell for Windows 3.1, featuring:

  • Shortcuts on the desktop
  • A taskbar with a Start Menu, Quick Launch, running programs, user-defined applets, and a clock
  • An easy-to-use replacement for File Manager
  • Windows 95 long filename support (if DOSLFN is installed)
  • 4DOS file description support
  • An implementation of the Recycle Bin
  • Multiple user support
  • Custom window borders

History

Calmira Reborn is fourth in the line of Calmira projects. It is a fork of Calmira LFN 3.32 by Alexandre Rodrigues de Sousa, itself a fork of Calmira II 3.3 by Calmira Online!, itself a fork of Calmira 2.2SR by Li-Hsin Huang.

This fork does not place much emphasis on new features and instead focuses on fixing issues I've discovered with Calmira LFN while using it on my old computers. So far, the following changes have been made:

  • Fix an instance of C:\WINDOWS being assumed as the Windows 3.1 installation directory
  • Fix numbered separators ("-1", "-2", etc.) displaying as menu items instead of separators when the Windows 3.1-style Start Menu is chosen
  • Fix the taskbar's free memory display from overflowing when 100 MB or more memory is available
  • Fix selection box appearing when multiple items can't be selected
  • Fix possible general protection fault when terminating a task via the taskbar
  • Fix various typos in the interface
  • Fix some resource leaks
