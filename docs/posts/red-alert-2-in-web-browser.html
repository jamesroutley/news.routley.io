Minimum/Recommended PC specifications

  • CPU: Intel Atom Z3700+ @1.33GHz (Intel Core i5 recommended)
  • OS: 64-bit operating system (and web browser)
  • Memory: 4GB (8GB recommended)
  • GPU: Intel HD Graphics (NVIDIA GTX800 or later series recommended)
  • Display resolution: 1024x768 minimum
  • Web browser: Latest Google Chrome, Mozilla Firefox, Microsoft Edge or Safari (for good performance, Firefox should be avoided)
