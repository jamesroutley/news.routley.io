@blueskiesoky Here is all what you need for a instant FeatherWiki Server

Download:

index.lua 
.init.lua
FeatherWikiRedBean.html
FWInstant.sh

In terminal:

cd ~/Downloads
chmod +x FWInstant.sh
./FWInstant.sh

That's it, should open a browser window with featherwiki in it. If for some reason the browser doesn't open, visit: http://localhost:8080 . Stop the server by pressing CTRL+C in Terminal.

More info:

Not recommended: Alternatively you can also just download FWInstant.sh and prebuilt FeatherWiki.com and do this:

cd ~/Downloads
chmod +x FeatherWiki.com
chmod +x FWInstant.sh
./FWInstant.sh

One can also manually start FeatherWiki.com

./FeatherWiki.com -M 5000000

To kill it:

killall -9 .ape

Explanation of files:
index.lua - Redbean FeatherWiki Saver
.init.lua - Starts Browser
FeatherWikiRedBean.html - Renamed FeatherWiki_Tern.html 1.3.1 version
FeatherWiki.com - Built FeatherWiki+Redbean Server 
FWInstant.sh - FeatherWiki Instant Runner.

How to build the tern / FeatherWikiRedBean.html version yourself?

Simple:

git clone https://codeberg.org/Alamantus/FeatherWiki.git
cd FeatherWiki
npm install
npm run build -- all
cd builds
cp FeatherWiki_Tern.html FeatherWikiRedBean.html

Security Note: Download and Run FeatherWiki.com server files only from sources that you trust.

Informative Note: All redbean downloads and self built / self compiled from source binaries have a entry on Virus Total due to antivirus detecting the APE multi OS as related to viruses. You can ignore those if you build/compile redbean yourself or download from trusted sources.

Hope this helps!
