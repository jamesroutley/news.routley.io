Open source smartwatch integrates ESP32 MCU with e-paper display

Apr 7, 2023 — by Giorgio Mendoza

3,493 views

ThePiHut featured this week an open source ePaper watch powered by an ESP32 module. Some of the peripherals featured include a 3-axis accelerometer, a vibration motor, a LiPo battery connector and many other features.

The Watchy has Bluetooth and Wi-Fi connectivity since it features the following microcontroller:

ESP32-PICO-D4 — Xtensa 32-bit LX6 MCU (up to 240MHz), 520 KB RAM, 4MB Flash

ThePiHut indicates that they are shipping the V2 version of the Watchy smartwatch, but it’s unclear what are the main differences from the version released on Crowdsupply in 2021. 

The Watchy comes with a 1.54” ePaper display with 200 x 200 resolution and a 24-pin FPC connector. There is also a 3-axis Bosch IMU optimized for wearable applications and with a motion-triggered interrupt.

         

The product page lists various documentation including a Watchy Wiki page and a GitHub repository. There is also a free simulator available to test new watch faces, refer to the following GitHub repository for more information.

         

Specification listed for the SQFMI Watchy:

  • Display:
    • 1.54” e-ink display (GDEH0154D67)
  • I/O Interfaces:
    • 4x Tactile buttons
  • Sensor:
    • 12-bit 3-Axis Accelerometer
  • USB:
    • 1x Micro USB port
  • Other Features:
    • RTC
    • Vibration Motor
    • Built-in USB-Serial adapter
  • Power:
    • 200mAh 3.7V LiPo Battery
    • 1x LiPo battery connector
  • Mechanical:
    • 34mm x 46mm
    • 18g

Further information

The SQFI Watchy is available for £57 on ThePiHut online store. The package contents include the Watchy PCB, 1x 3.7v LiPo battery, 1x Watchy injection-molded case, 1x black strap, 1x ePaper display and 1x double sided tape.
