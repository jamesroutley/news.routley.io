We Fund Public Interest Tech

The Prototype Fund supports ideas in civic tech, data literacy, data security and software infrastructure. We want to provide software developers, hackers, and creatives with the opportunity to code and develop innovative public interest technologies.

The Prototype Fund is the first and only funding program of its sort in Germany. We provide the place and the means necessary to implement innovative software prototypes with a benefit for society: Especially for freelance developers and interdisciplinary teams that have not been considered by government funding instruments before.

With this funding, they can write code and develop a first prototype of your software. In addition, we will connect them with tech and other relevant communities, support them with coaching, and advise them on project implementation and communication.

Only applicants with a residency in Germany are eligible for funding. If you are not a resident in Germany but interested in the funding program nevertheless take a look at our funded projects, their code base, funding criteria and the FAQ. If you have any questions or plan to set up a similar funding program feel free to contact the team. We are happy to help!
