Show HN: Visual Sudoku solver in the browser

Welcome to this Visual Sudoku Solver

This app lets you solve Sudoku puzzles by simply pointing your camera at them. It's a project I built to learn and experiment with computer vision and machine learning.

How to use

  1. Point your camera at the Sudoku puzzle.
  2. Make sure the whole grid is visible in the camera frame.
  3. Hold still for a few seconds while the app works its magic and fills in the missing numbers.

Tips for best results

  • Lighting
  • Angle
  • Focus

Known issues

I built this mainly for my own learning, so it's definitely not perfect. Some more common issues are:

  • Sometimes, the app has trouble recognizing the grid, especially if the lighting is uneven or the puzzle isn't clear.
  • It might occasionally misread parts of the grid, leading to some errors.

Learn more

If you'd like to learn more about how I built this, I documented a lot of the progress in python notebooks which are available on Github.
