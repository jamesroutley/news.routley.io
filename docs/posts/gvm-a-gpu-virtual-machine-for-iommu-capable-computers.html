GVM: A GPU Virtual Machine for IOMMU-Capable Computers

Introduction

GVM is a GPU Virtual Machine for IOMMU-capable computers such as x86 and ARM.

It's comprised of several components including: kernel modules, userspace libraries, cli tools, and daemons.

As of July 6th 2022 the first GVM component (mdev-gpu) is made available as GPLv2.

mdev-gpu 0.1.0.0 includes the following:

Current Release Functions

At present the following features are supported:

  • Creation of arbitrary GPU mdevs based on user defined properties.

  • Support for Nvidia GPUs.

Planned Release Functions

Future releases may include:

  • Support for Intel GPUs.

  • Support for Tenstorrent TPUs.

Documentation

Documentation on mdev-gpu API can be found at https://docs.linux-gvm.org/mdev-gpu/

Formal verification (TLA+) for OpenRM driver interactions can be found at https://github.com/Arc-Compute/LibRM/

Comprehensive Mediated Device Internals documentation can be found at https://openmdev.io/index.php/Mediated_Device_Internals

Source Code

Source code for mdev-gpu can be found at https://github.com/Arc-Compute/Mdev-GPU/
