J2ME-Loader: J2ME emulator for Android devices

J2ME-Loader is a J2ME emulator for Android. It supports most 2D and 3D games (including Mascot Capsule 3D ones). Emulator has a virtual keyboard, individual settings for each application, scaling support. This project is a fork of J2meLoader.

System requirements: Android 4.0+

List of the tested Java Games (Touchscreen)

  • Enabling filtering in some cases can greatly reduce performance. Disable this option if game is too slow.
  • Image flickering issues can be fixed by enabling the "Immediate processing mode" option.
