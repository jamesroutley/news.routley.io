Microsoft is killing custom domain names in Outlook.com

One of the features premium Outlook.com users have access to is the ability to host a custom domain name on Outlook.com and therefore to have personalized email addresses on there.

Microsoft notes:

As a benefit of your Microsoft 365 Family or Microsoft 365 Personal subscription, you can create a personalized email address that’s associated with your Outlook.com mailbox. For example, yourname@example.com. 

Unfortunately, it appears this feature is going away, according to a message Microsoft has been sending out to subscribers.

It states:

• Personalized email addresses in Outlook.com

Fortunately the change will not affect existing users, and it appears current subscribers have around 4 weeks to add their own custom domain before they will no longer be unable to, as availability is being removed on the 30th November.

Microsoft said:

If you already have a personalized email address set up, you will not be impacted by this change, and you can continue using it without any disruption in service. After November 30, 2023, if you choose to remove your existing personalized email address, you will be unable to add it back.

If you want to take advantage of the feature before Microsoft withdraws it, read how at Microsoft’s FAQ here.
