CUBOTino: A small, simple, 3D printed, inexpensive Rubik's Cube solver robot

This repo contains the files to build CUBOTino_base_version: a Small, Simple, 3D Printed, Inexpensive Rubik's Cube Solver Robot.

This robot solves the Rubik's cube in less than 90 seconds: Not fast, but again ... it's rather simple and inexpensive.

All the needed info are collected in a pdf file in the /doc folder: document here.

  • 3D print without support.
  • Set the servo to their mid position.
  • Verify the servos have 180deg range.
  • For the rotating base (cube_holder) the servo must have at least 190deg rotation; If not check the instruction on how to proceed.
  • If you use macOS or Ubuntu, update RubikTwoPhase solver to latest version (v1.1.1, released on 16th Nov. 2022).

Video tutorial explaining how to present the cube to the webcam: https://youtu.be/udr6tryxA_Y.

The movie is also available at /movies folder of this repo, and can be downloaded.

In case you find this robot too simple, or you're interested to a self-enclosed version, then you can consider the CUBOTino Autonomous version:

  • It reuses most of the parts from this Base version !!!
  • Thanks to a camera and a SBC (Raspberry Pi) the robot if fully autonomous.
  • You can see it at: https://youtu.be/udr6tryxA_Y.

I hope many of you will decide to build your own CUBOTino, and that you'll enjoy it as much much as I did.
