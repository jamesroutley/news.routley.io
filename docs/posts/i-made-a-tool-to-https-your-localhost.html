Show HN: I made a tool to HTTPS your localhost

Local dev are supposed to be simple and easy.

Supercharge your localhost, all tools at your fingertips.

Self-hosted Lokal Tunnel Server

Freedom and Privacy at once, Deploy on VPS or Bare-metal with Docker.

Cloudflare Ephemeral Tunnel

Leverage Cloudflare 310+ Location to speed up your site delivery.

AI Assistant

GPT-4o powered Assistant, Interact with tunnel traffic, generate codes, ask anything related to the traffic content.

Unlimited .local Domain

LAN-Wide Accessible .local address, https by default, no traffic hit Lokal server, perfect for non-remote team.

S3 Built-in Server

Say goodbye to docker, Live S3 Server and SLOW S3 Browser GUI!, Lokal provide built-in S3 Compatible server that is leveraging User's File System for security and easy file debugging!

JSON to Language Scheme

Automatically convert JSON Payload from your request body into any your programming language model.

Inspect, Modify, Replay

Keep it clean, no more print() for debugging request and response content.

Endpoint Management

Stop memorizing your IPs and ports address, use memorable names instead!

Web, Desktop and CLI

Developer Experience is our first priority, Lokal ship with Web, Desktop and CLI interface, choose one for your use-case.

JSON Prettifier & Finder

Read what you need to read, see what you need to see, built-in JSON Prettifier and JSON Query based on JMESPath.

File Server

Easily share your local folder with others, add Authentication to make sure only intended people has access to.
