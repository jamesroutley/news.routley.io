Woah, Eruda

Today I found out about eruda. It makes me feel silly for writing up this post on usb debugging iOS Safari a few months back.

Eruda embeds a debugging overlay into a website. It provides a little floating icon that can be tapped to activate it. The overlay has all the usual suspects - DOM inspector, network logs, and most valuable to me: the console log. No need to wire up my phone to my computer and get a remote inspector to work, which I've found to be a particularly finicky process given my iPhone's lint-filled lightning port.

Just check it out: eruda. It is dead simple to install.
