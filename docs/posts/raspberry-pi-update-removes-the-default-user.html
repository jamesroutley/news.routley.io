Raspberry Pi update removes the default user

New Raspberry Pi OS April update removes the default Pi user

Why have these changes been made?

According to the blog post the changes have been due to new legislation forbidding default users in the software. One of these countries is the Raspberry Pi Foundations home country, England. Companies can be fined up to 10 Million pounds or 4% of their global revenue, as well as 20 thousand pounds a day. The law was introduced to reduce the number of cyberattacks on unsecured devices with default credentials. You can find more information about this legislation here: https://www.bbc.com/news/technology-59400762

How to set up a new user with Raspberry Pi Imager

  1. Download Raspberry Pi Imager from the Raspberry Pi Website:
  2. Execute the downloaded file and install it
  3. Launch the Raspberry Pi Imager
  4. Press the “choose OS” button and choose an OS like Raspberry Pi OS
  5. Press the settings gear in the lower right of the Imager
  6. scroll down to the set username and password screen
  7. confirm with the save button

The current version of the Imager still allows you to set the old default credentials

You can also create a user in the setup menu when starting your new pi. The imager also allows you to connect your pi to a WIFI and setup an SSH connection headless.
