Even the touch screen works like a charm â€“ the same as my other ThinkPad X220t (tablet) â€¦ and even no additional configuration was needed â€“ I just used the configuration that I use daily on my ThinkPad W520 laptop. From many things that I really like about FreeBSD (more here â€“ Quare FreeBSD? â€“ in separate article) is that it can be entirely configured using just 3 files. This configuration already features all power management settings that I described in the The Power to Serve â€“ FreeBSD Power Management article.

I installed FreeBSD in a pretty standard way with GELI encryption enabled and with ZFS as the filesystem. When in doubt the installation procedure is described in the FreeBSD Desktop â€“ Part 2.1 â€“ Install FreeBSD 12 article.

  Option "NaturalScrolling" "on"
