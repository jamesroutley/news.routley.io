Show HN: Scipress.io – a Markdown writing platform like Notion Medium MkDocs

Scipress is a platform for writing and sharing content. It's geared towards technical writers like programmers, mathematicians, engineers, and scientists, but it can be used by anyone.

Everything you see here was written in Scipress!

Scipress takes inspiration from tools and platforms like

  • Docusaurus
  • Material for MkDocs
  • Substack
  • Medium
  • StackOverflow
  • Reddit

Why should I use Scipress?¶

  • Scipress makes writing technical content fun and easy.
  • You can earn money by selling access to gated content.
  • It's free. Only pay if and when you earn.
  • It has a rich set of features for organizing and designing content.
  • It's beautiful, by default

What can I make with Scipress?¶

With Scipress, you could make...

  • A recipe book where your best recipes are gated behind a paywall

  • A private, internal knowledgebase

  • A collection of How To manuals

  • A collection of programming challenges
