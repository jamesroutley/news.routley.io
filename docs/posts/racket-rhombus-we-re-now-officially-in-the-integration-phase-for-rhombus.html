We're now officially in the "integration" phase for Rhombus.

This repo has been renamed to racket/rhombus, but racket/rhombus-prototype redirects here. The repo content has been split into multiple packages, but the rhombus-prototype package (not meant to be used going forward) implies the initial set of Rhombus packages, for your transition convenience.

In this new phase, we start building out Rhombus libraries. That will include documentation, and Rhombus Scribble does exist (and is used to document the Rhombus language). But Rhombus Scribble is not itself documented, and I'd like to take the opportunity to further improve documentation support and use it to document Rhombus Scribble in the coming days or weeks.

I expect that we'll want to create a list of libraries to be adapted/ported. Let's discuss that at the July 25 meeting (#180).
