Show HN: A VNC viewer for eInk devices capable of 30 FPS when writing text

A VNC client optimised for eInk displays

A lightweight CLI (command line interface) tool to view a remote screen over VNC, designed to work on eInk screens. For now, you can only view, so you'll have to connect a keyboard to the serving computer, or find some other way to interact with it.

This tool has been confirmed to work on the Kobo Libra 2, and should work on all Kobo devices. It was optimized for text based workflows (document reading and writing), doing that it achieves a framerate of 30 fps.

It has only been confirmed to work with TightVNC as the server. Due to the unusual pixel format.

A warning

The screen can refresh up to 30 times per second, this will degrade the eInk display rapidly. Do not use with fast changing content like videos.

Furthermore, this tool was only tested on a single device (Kobo Libra 2). It is possible that it will damage yours. I cannot be held responsible, use this tool at your own risk.

Usage

You can use this tool by connecting to the eInk device through SSH, or using menu launchers on the device itself.

To connect to a VNC server:

./einkvnc [IP_ADDRESS] [PORT] [OPTIONS]

For example:

./einkvnc 192.168.2.1 5902 --password abcdefg123 --contrast 2

For faster framerates, use USB networking (see this thread).
