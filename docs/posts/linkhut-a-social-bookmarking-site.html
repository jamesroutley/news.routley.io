日向坂46 plays 大富豪 (a “The Great Dalmuti” style game). The tournament starts after two songs.

They’re using a french deck and this ruleset: taxation, big two, jokers are wild, a single joker beats any other single, both sets and runs are fine, and you need to match type to type. No bombs. A pretty good ruleset if you don’t have the pyramidal “Dalmuti” deck!
