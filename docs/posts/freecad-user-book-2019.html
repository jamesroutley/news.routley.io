FreeCAD User Book (2019)

About This Book

This book is a documentation of my journey, learning Freecad. It is not a linear path from beginning to end, the sequence of topics added may appear somewhat randomly. For instance, I started with Realthunder's Assembly 3 Preview branch of Freecad, which offered the Link feature, an object that supports re-use of components without duplicating their definition, local or across file boundaries. I found that appealing for organizing standard parts in repositories.
