Superhighway84 – a Usenet-inspired decentralized internet discussion system

Superhighway84 is an USENET-inspired decentralized internet discussion system, featuring a retro text user interface and powered by IPFS and OrbitDB.

With the emerge of all different kinds of Omniverses and Metaverses I decided to counter these trends by bringing back the good old days of communication via simple text interfaces – but with a modern twist.

Superhighway84 is using berty’s Go implementation of OrbitDB, which allows it to maintain a publicly writeable database that is shared across all participating peers over IPFS. Its text user interface is powered by the great tview library and is kept intentionally minimal and simple. For reading and writing articles it integrates with your preferred $EDITOR, so that you can easily integrate Superhighway84 into your existing workflow.

Superhighway84 is open source software and can be downloaded from GitHub. Pre-built releases are available for Linux, OpenBSD, FreeBSD, NetBSD, Plan9, macOS and Windows.
