we are a community that exists on a linux server. founded 2014. run by ~vilmibm. population around 1,000.

stuff we like:

  • empathy
  • making art
  • creative use of technology
  • using old things to do new things
  • using new things to do weird things
  • socializing
  • peer education

if you think you might enjoy being a part of this community you can apply for a homestead using this form.

it can take up to 30 days to be approved and not all applications are approved.

if you are new to ssh, there is some help with it here

users make webpages sometimes. you can check them out on this page

a blogthing chronicles system updates. you can read it. it lives here.

the town is a non-commercial thing. donations are appreciated

are you a town resident that lost their ssh key? try this:

  • using the email address with which you registered
  • send an email to root@tilde.town
  • put "new public key" in the subject
  • include the new public key in the body of the email
  • wait for an admin to reply and confirm the update
