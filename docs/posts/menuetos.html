MenuetOS

MenuetOS is an operating system for PC, written fully in assembly language (64bit and 32bit). Features include pre-emptive and real-time multitasking with multiprocessor support and Graphical User Interface. Menuet64 is released under License and Menuet32 under GPL. Menuet supports both 64 and 32bit x86 assembly programming for smaller, faster and less resource hungry applications.
