Hacker News in Slow Italian

I've been doing Duolingo for Italian for a while, but still struggle to use the language in practice. Can generative AI help me and anyone else stuck in this Duolingo tranche?Each episode is generated automatically, using GPT4 API to summarise the top 5 articles on Hacker News and Play.ht for text-to-speech. Check out the code on Github.The key concept is that generative AI can help language learning become targeted, immediate, and interactive. There are podcasts in Italian, but none is targeted to the topics I'm interested in and uses a beginner-friendly language I can undersand.
