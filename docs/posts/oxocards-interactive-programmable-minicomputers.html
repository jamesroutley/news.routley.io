Oxocards: Interactive programmable minicomputers

The Oxocards are interactive programmable minicomputers, with extensive material.

Suitable from 14 years.

Oxocard Science is programmed with the text-based scripting language Oxoscript. Extensive tutorials are available, which can be used for self-study as well as for teaching.

Suitable from 14 years.

The Oxocard Artwork is programmed with the text-based scripting language Oxoscript. Extensive tutorials are available, which can be used for self-study as well as for teaching, as well as in the classroom.

Learn the concepts of programming with the graphical programming language "Blockly". Step by step you will learn and implement the more complex concepts.

Suitable from 10 years.

The Oxocard is programmed with the intuitive and fault-tolerant block programming environment "Blockly". Thus, this course can also be successfully used in class by teachers without programming knowledge.
