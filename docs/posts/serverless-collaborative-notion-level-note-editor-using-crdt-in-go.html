File manager + CRDT Note Editor

  • File manager (inspired by LF)
  • nb-editor enables nb-crdt(nb-crdt-go) synchronization and collaboration just through a shared folder, eliminating the need for a server.

Development

Prerequisites

Install go, npm, platform-specific dependencies, and wails by referring to wails installation.

Run

wails dev

Key Shortcuts

m > n (new note)
?     (open manual)
