"cstring" is an abstraction for handling C-style strings in JavaScript (null-terminated, so they end with a \0) you can use them like a regular string 6 / 8

To make using pointers easier with native libraries, bun:ffi exports a "ptr(typedArray)" and "toArrayBuffer(aPtr) functions it also converts typed arrays automatically when passing to FFI functions 7 / 8
