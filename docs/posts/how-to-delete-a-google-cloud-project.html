How to Delete a Google Cloud Project

DISCLOSURE: If you buy through affiliate links, I may earn a small commission. (disclosures)

In this post we walk through how to delete a Google Cloud Project.

I personally moved all my projects off Google Cloud last year to self-hosted and finally got around to cleaning up all my dangling resources over on Google Cloud.

To delete a Google Cloud Project

  • Log into Google Cloud
  • Select the Project Picker in the top left
  • Click the hamburger menu / 3 dots in the top right and select Manage resources
  • Select the Project row you want to delete (you can only delete one project at a time)
  • Once selected, the Delete button should enable in the top row. Click that.
  • This will pop up a large warning giving you details about deletion and warning you about any activity in the project over the last 30 days so you can ensure you're really done with it
  • Once you're sure, scroll down and type in your project-id to the text box and select Shut down anyway

If everything goes well, you should see a toast saying your project is being deleted followed by a modal confirming project deletion.
