Show HN: A tool to analyze Hacker News sentiment on any term in seconds

How to use hackernews sentiment analyzer:

  1. Enter an exact phrase (e.g., "founder mode", "remote work") to see a chart of its sentiment on hackernews over the past month
  2. Responses contain a random sample from the comments, not AI-generated
  3. Each message is independent with no conversation history
  4. Searches are limited to 5000 datapoints -- create an account for unlimited requests with the API
  5. Popular phrases (e.g., "openai", the letter "e") will take longer to pull data for and analyze
